# Decor Gramas

Site institucional da Decor Gramas — venda e instalação de grama sintética premium no Vale do Itajaí (SC).

## Tecnologia

Site estático em HTML/CSS/JS puro. Sem dependências de build, sem framework. Deploy direto do diretório raiz.

## Estrutura

```
├── index.html          Home
├── servicos/           Páginas de serviço
├── assets/
│   ├── css/            Design system compartilhado
│   ├── js/             Comportamentos compartilhados
│   ├── brand/          Logos e favicons
│   └── images/         Imagens
├── robots.txt
└── sitemap.xml
```

## Serviços cobertos

- Grama sintética para jardim
- Grama sintética decorativa
- Grama sintética para pets
- Grama sintética para piscina
- Grama sintética para varanda
- Playground em grama sintética

## Deploy

Copiar todos os arquivos para o servidor web ou arrastar para o painel do host. Não requer nenhum processo de build.
